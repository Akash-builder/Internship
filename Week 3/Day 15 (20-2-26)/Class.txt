1) What is Matplotlib? Explain about it
Matplotlib is a popular Python library used for data visualization. It helps in creating static,
animated, and interactive plots such as line charts, bar charts, histograms, scatter plots, pie
charts, and more.
It is widely used in:
• Data analysis
• Machine learning
• Scientific computing
• Business reporting
Matplotlib converts numerical data into graphical representation, making it easier to identify
trends, patterns, correlations, and outliers.
It was introduced by John D. Hunter in 2003.

2) Difference between plot() and scatter()
plot() scatter()
Used mainly for line graphs Used for scatter plots
Connects points with lines Shows only individual points
Best for trends over time Best for correlation between two variables
Example:
plt.plot(x, y) # Line plot
plt.scatter(x, y) # Scatter plot

3) What is Pyplot? Explain about it
pyplot is a submodule of matplotlib.
It provides a MATLAB-like interface to create plots easily.
It is usually imported as:
import matplotlib.pyplot as plt
Pyplot contains functions like:
• plot()
• scatter()
• bar()
• hist()
• xlabel()
• ylabel()
• title()
• show()
It makes plotting simple and interactive.

4) How to add labels and title
We use:
• plt.xlabel("X-axis name")
• plt.ylabel("Y-axis name")
• plt.title("Graph Title")
Example:
plt.plot(x, y)
plt.xlabel("Months")
plt.ylabel("Sales")
plt.title("Monthly Sales Trend")
plt.show()

5) Difference between Figure and Axes
• Figure → Entire canvas or window that holds the plot
• Axes → The actual graph area inside the figure
Think of:
• Figure = Paper
• Axes = Drawing on the paper
Example:
fig, ax = plt.subplots()
ax.plot(x, y)

6) How to create multiple subplots?
Using subplot() or subplots():
plt.subplot(2, 1, 1)
plt.plot(x, y)
plt.subplot(2, 1, 2)
plt.plot(a, b)
plt.show()
Or:
fig, ax = plt.subplots(2, 2)

7) Use of Marker & Different Markers
Markers highlight data points in a graph.
Example:
plt.plot(x, y, marker='o')
Common markers:
Marker Shape
'o' Circle
'*' Star
Marker Shape
'.' Point
'p' Pentagon
'd' Diamond
's' Square
'^' Triangle

8) How to Customize Colors?
Using color parameter:
plt.plot(x, y, color='red')
Short color codes:
• 'r' → Red
• 'g' → Green
• 'b' → Blue
• 'k' → Black
• 'y' → Yellow
We can also use HEX colors:
plt.plot(x, y, color='#FF5733')

9) How to handle large amounts of data in visualization?
• Use sampling (plot subset of data)
• Use alpha (transparency)
• Use smaller markers
• Use aggregation (groupby)
• Use histogram instead of scatter for distribution
• Use libraries like seaborn or plotly for interactive plots
Example:
plt.scatter(x, y, alpha=0.3)

10) Which plot is used for trends in data?
Line Plot (plot())
Best for:
• Time series

• Sales trends
• Performance tracking

11) Which plot is used for data distribution?
• Histogram → Shows frequency distribution
• Boxplot → Shows spread and outliers
Example:
plt.hist(data)

12) How to change line color?
Using:
plt.plot(x, y, color='blue')

13) Different line styles available
Style Meaning
'-' Solid line
'--' Dashed line
':' Dotted line
'-.' Dash-dot line
Example:
plt.plot(x, y, linestyle='--')

14) How to display grid?
Using:
plt.grid()
Customize grid:
plt.grid(linestyle='--', color='gray')
